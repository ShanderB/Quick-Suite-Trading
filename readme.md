## Ferramentas utilizadas!

- Node 16.15.0
- Nvm 1.1.11
- Npm 8.5.5


## Como rodar o projeto?
Tenha certeza de que está utilizando as versões corretas.

Basta instalar o projeto:

    npm i

Após finalizado, inicie o serviço:

    npm start

O projeto irá iniciar no link:

    http://localhost:4200/
