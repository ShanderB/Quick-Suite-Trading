## Tools used!

- Node 16.15.0
- Nvm 1.1.11
- Npm 8.5.5


## How to run the project?
Make sure you are using the correct versions.

Enter the webapp directory

    cd webapp-viso

Just install the project:

    npm i

Once finished, start the service:

    npm start

The project will start at the link:

    http://localhost:4200/
